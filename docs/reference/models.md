# Models

The Jules API defines various data models that are mapped to Python objects by the SDK.

## Session

Represents a session in the Jules API.

*   **`name`**: (`str`) The unique name of the session.
*   **`state`**: (`SessionState`) Current execution state of the session.
*   **`create_time`**: (`str`) When the session was created.
*   **`update_time`**: (`str`) When the session was last updated.
*   **`id`**: (`str`, optional) The ID of the session.
*   **`title`**: (`str`, optional) The title.
*   **`require_plan_approval`**: (`bool`, optional) Whether the plan needs approval.
*   **`source_context`**: (`SourceContext`, optional) The context for the session.
*   **`prompt`**: (`str`, optional) The prompt provided to Jules.
*   **`automation_mode`**: (`AutomationMode`) Automation setting. Defaults to `AUTOMATION_MODE_UNSPECIFIED`.
*   **`outputs`**: (`List[SessionOutput]`) Execution outputs.
*   **`archived`**: (`bool`) Whether the session is archived.
*   **`url`**: (`str`, optional) A URL related to the session.

## SourceContext

Specifies the context in which Jules operates.

*   **`source`**: (`str`) The source identifier.
*   **`github_repo_context`**: (`GitHubRepoContext`, optional) Configuration for a GitHub repository.
*   **`working_branch`**: (`str`, optional) Working branch.
*   **`environment_variables_enabled`**: (`bool`, optional) Whether to use env vars.

## GitHubRepoContext

Specifies a GitHub repository for a session.

*   **`github_repo`**: (`GitHubRepo`, optional) The repository.
*   **`starting_branch`**: (`str`, optional) Starting branch.

## Source

Represents a repository source.

*   **`name`**: (`str`) The name of the source.
*   **`id`**: (`str`) The ID of the source.
*   **`github_repo`**: (`GitHubRepo`, optional) GitHub repository details.

## Activity

Represents an event that happened within a session.

*   **`name`**: (`str`) The name of the activity.
*   **`create_time`**: (`str`) The creation time.
*   **`type`**: (`ActivityType`) The type of activity.
*   **`id`**: (`str`, optional) ID of the activity.
*   **`description`**: (`str`, optional) A description.
*   **`originator`**: (`str`, optional) The entity that created the activity.
*   **`artifacts`**: (`List[Any]`) Related artifacts.
*   **`details`**: (`Dict[str, Any]`) Additional type-specific details.

## Plan

Represents an execution plan generated by Jules.

*   **`id`**: (`str`) Plan ID.
*   **`steps`**: (`List[PlanStep]`) Individual steps.
*   **`create_time`**: (`str`) When the plan was created.

## PlanStep

A single step in a plan.

*   **`id`**: (`str`) The step ID.
*   **`title`**: (`str`) A brief title.
*   **`description`**: (`str`) The detailed description.
*   **`index`**: (`int`) Execution index of the step.

## SessionOutput

Output from a session execution.

*   **`pull_request`**: (`PullRequest`, optional) Created pull request details.
*   **`change_set`**: (`ChangeSet`, optional) Git patches.

## PullRequest

Details of a pull request.

*   **`url`**: (`str`) The PR URL.
*   **`title`**: (`str`) The title.
*   **`description`**: (`str`) The PR description.
*   **`base_ref`**: (`str`) Base reference.
*   **`head_ref`**: (`str`) Head reference.

## ChangeSet

Code changes generated by the session.

*   **`git_patch`**: (`GitPatch`) The actual patch.
*   **`source`**: (`str`) Source.

## GitPatch

Details of a git patch.

*   **`unidiff_patch`**: (`str`) Diff output.
*   **`base_commit_id`**: (`str`) The commit ID.
*   **`suggested_commit_message`**: (`str`) A generated commit message.

## GitHubRepo

GitHub repository information.

*   **`owner`**: (`str`) Repository owner.
*   **`repo`**: (`str`) Repository name.
*   **`is_private`**: (`bool`) Whether it's private.
*   **`default_branch`**: (`GitHubBranch`) Default branch info.
*   **`branches`**: (`List[GitHubBranch]`) Available branches.

## GitHubBranch

Information about a GitHub branch.

*   **`display_name`**: (`str`) Display name.
