# Models

The Jules API defines various data models that are mapped to Python objects by the SDK.

## Session

Represents a session in the Jules API.

*   **`name`** (`str`): The unique name of the session, usually structured like `sessions/{uuid}`.
*   **`state`** (`SessionState`): Current execution state of the session.
*   **`create_time`** (`str`): RFC3339 formatted string indicating when the session was created.
*   **`update_time`** (`str`): RFC3339 formatted string indicating when the session was last updated.
*   **`id`** (`str`, optional): The ID of the session. Defaults to `""`.
*   **`title`** (`str`, optional): A human-readable title for the session.
*   **`require_plan_approval`** (`bool`, optional): Whether the session will wait in `AWAITING_PLAN_APPROVAL` state before executing a plan.
*   **`source_context`** (`SourceContext`, optional): The context detailing where the session operates.
*   **`prompt`** (`str`, optional): The task description provided to Jules.
*   **`automation_mode`** (`AutomationMode`): Automation setting determining if PRs are auto-created. Defaults to `AUTOMATION_MODE_UNSPECIFIED`.
*   **`outputs`** (`List[SessionOutput]`): Execution outputs containing items like `ChangeSet` or `PullRequest` objects. Defaults to an empty list.
*   **`archived`** (`bool`): Whether the session is archived. Defaults to `False`.
*   **`url`** (`str`, optional): A URL linking to the web interface for the session.

## SourceContext

Specifies the context in which Jules operates.

*   **`source`** (`str`): The source identifier (e.g. `sources/github/owner/repo`).
*   **`github_repo_context`** (`GitHubRepoContext`, optional): Additional configuration details when operating on a GitHub repository.
*   **`working_branch`** (`str`, optional): The name of the working branch for changes.
*   **`environment_variables_enabled`** (`bool`, optional): Whether to inject environment variables into the session.

## GitHubRepoContext

Specifies a GitHub repository for a session.

*   **`github_repo`** (`GitHubRepo`, optional): The repository object with owner, repo, and branch information.
*   **`starting_branch`** (`str`, optional): The branch from which Jules will branch off.

## Source

Represents a repository source.

*   **`name`** (`str`): The unique name of the source (e.g. `sources/{id}`).
*   **`id`** (`str`): The UUID or ID of the source. Defaults to `""`.
*   **`github_repo`** (`GitHubRepo`, optional): GitHub repository details.

## Activity

Represents an event that happened within a session.

*   **`name`** (`str`): The unique name of the activity, often `sessions/{session_id}/activities/{id}`.
*   **`create_time`** (`str`): RFC3339 formatted string indicating the creation time.
*   **`type`** (`ActivityType`): The type of activity derived from its details (e.g. `AGENT_MESSAGED`).
*   **`id`** (`str`, optional): ID of the activity. Defaults to `""`.
*   **`description`** (`str`, optional): A human-readable description.
*   **`originator`** (`str`, optional): The entity that created the activity (e.g. `USER`, `SYSTEM`, `AGENT`).
*   **`artifacts`** (`List[Any]`): Related artifacts attached to the activity. Defaults to an empty list.
*   **`details`** (`Dict[str, Any]`): A dictionary representing the type-specific details payload (e.g. the actual message). Defaults to an empty dictionary.

## Plan

Represents an execution plan generated by Jules.

*   **`id`** (`str`): Unique plan ID.
*   **`steps`** (`List[PlanStep]`): Ordered list of individual steps required to complete the plan. Defaults to empty list.
*   **`create_time`** (`str`): RFC3339 formatted string indicating when the plan was created.

## PlanStep

A single step in a plan.

*   **`id`** (`str`): The unique step ID.
*   **`title`** (`str`): A brief, descriptive title for the action to be taken.
*   **`description`** (`str`): The detailed description of the execution step.
*   **`index`** (`int`): Execution order index of the step within the plan.

## SessionOutput

Output from a session execution.

*   **`pull_request`** (`PullRequest`, optional): Created pull request details, if `automation_mode` or action triggered PR creation.
*   **`change_set`** (`ChangeSet`, optional): Git patches reflecting code modifications.

## PullRequest

Details of a pull request.

*   **`url`** (`str`): The GitHub URL pointing to the PR. Defaults to `""`.
*   **`title`** (`str`): The title of the PR. Defaults to `""`.
*   **`description`** (`str`): The body description of the PR. Defaults to `""`.
*   **`base_ref`** (`str`): The base branch reference (e.g. `main`). Defaults to `""`.
*   **`head_ref`** (`str`): The head branch reference containing the changes. Defaults to `""`.

## ChangeSet

Code changes generated by the session.

*   **`git_patch`** (`GitPatch`): The actual patch information. Defaults to empty patch.
*   **`source`** (`str`): Source reference for the changes. Defaults to `""`.

## GitPatch

Details of a git patch.

*   **`unidiff_patch`** (`str`): The unified diff output of changes. Defaults to `""`.
*   **`base_commit_id`** (`str`): The base commit ID hash from which changes were made. Defaults to `""`.
*   **`suggested_commit_message`** (`str`): A generated commit message summarizing the patch. Defaults to `""`.

## GitHubRepo

GitHub repository information.

*   **`owner`** (`str`): GitHub repository owner or organization. Defaults to `""`.
*   **`repo`** (`str`): GitHub repository name. Defaults to `""`.
*   **`is_private`** (`bool`): Whether the repository is private. Defaults to `False`.
*   **`default_branch`** (`GitHubBranch`): Default branch info. Defaults to empty branch.
*   **`branches`** (`List[GitHubBranch]`): Available branches. Defaults to an empty list.

## GitHubBranch

Information about a GitHub branch.

*   **`display_name`** (`str`): The display name of the branch.

## AutomationMode

Enum detailing automation options for session outputs.
*   **`AUTOMATION_MODE_UNSPECIFIED`**: No specific mode assigned.
*   **`AUTO_CREATE_PR`**: Automatically creates a Pull Request when execution finishes.

## SessionState

Enum detailing the current state of a Session.
*   **`STATE_UNSPECIFIED`**
*   **`CREATED`** (obsolete, retained for compatibility)
*   **`QUEUED`**
*   **`RUNNING`** (obsolete, retained for compatibility)
*   **`IN_PROGRESS`**
*   **`PAUSED`**
*   **`AWAITING_USER_FEEDBACK`**
*   **`COMPLETED`**
*   **`FAILED`**
*   **`CANCELLED`** (obsolete, retained for compatibility)

## ActivityType

Enum detailing the type of `Activity`.
*   **`AGENT_MESSAGED`**
*   **`USER_MESSAGED`**
*   **`PLAN_GENERATED`**
*   **`PLAN_APPROVED`**
*   **`PROGRESS_UPDATED`**
*   **`SESSION_COMPLETED`**
*   **`SESSION_FAILED`**
